# Prezenta-QR
Aplicatie de gestionare a prezentelor

-Scanarea codului QR se va face cu ajutorul camerei foto a telefonului (Logica va fi browser based)



Tehnologii:

-Java Spring REST API.

-REACT.js

-Postgre SQL/Microsoft SQL/My SQL WorkBench
